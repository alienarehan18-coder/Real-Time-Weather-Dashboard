# Real-Time-Weather-Dashboard

**Description**
A real-time weather dashboard that fetches live weather data using the OpenWeatherMap API. Users can search for any city and view temperature, humidity, wind speed, and weather conditions.

**Features**

Live API integration

Search functionality

Weather icons

Loading state handling

Responsive UI

**Technologies Used
**
HTML

CSS

JavaScript

OpenWeatherMap API

Developed using VS Code
